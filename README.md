# pals
Production Automatic Logging System (PALS) is a sophisticated production floor software that generates and tracks Batch IDs for small to medium scale factories. PALS utilizes an End-to-End Traceability Batch ID System format to track products, guaranteeing each product's manufacturing process is recorded along the way.

## Features
- Clean user interface with intuitive buttons
- Accessible database via Google Sheets
- Secure login system with customizable passkeys
- Automatic generation of batch IDs and tracking
- Dashboard to view all current and past batches
